Создайте интерфейс веб-страницы, который включает в себя следующие элементы: Выпадающий список (select) с категориями товаров. Список товаров, который будет отображать товары в соответствии с выбранной категорией. Каждый товар в списке должен содержать название и категорию.

При выборе категории товаров в выпадающем списке, форма должна следить за изменениями. Динамически обновите список товаров на странице, чтобы отображать только товары из выбранной категории.
